📰 RSS 阅读器 - 使用说明

⚠️ 重要提示：CORS 跨域问题
由于浏览器安全限制，直接从网页访问其他网站的 RSS 源会被 CORS 策略阻止。 本阅读器使用公共 CORS 代理来解决此问题。

🚀 快速开始

方案 1：直接使用（推荐）
双击 index.html 打开，系统会自动使用公共 CORS 代理（api.allorigins.win）。
✅ 优点：无需安装任何东西
⚠️ 缺点：依赖第三方代理服务
方案 2：使用本地 HTTP 服务器（最佳体验）
cd /Users/weizhao/Desktop/rss
python3 -m http.server 8000
然后在浏览器访问：http://localhost:8000
✅ 优点：性能最好，无第三方依赖
方案 3：安装 CORS 浏览器插件（开发调试用）
Chrome 插件：Allow CORS: Access-Control-Allow-Origin
✅ 优点：可访问任何 RSS 源
⚠️ 缺点：需要安装插件

📁 文件结构

rss/
├── index.html      # 主页面
├── css/
│   └── style.css   # 样式文件
├── js/
│   └── app.js      # 应用逻辑（纯前端解析）
├── opml/
│   └── feeds.opml  # 订阅源配置
└── README.html     # 本说明文档

➕ 添加新订阅源

编辑 opml/feeds.opml 文件，在 <body> 内添加：

<outline text="网站名称" title="显示名称" type="rss" 
         xmlUrl="RSS 地址" htmlUrl="网站地址"/>

🔧 技术说明

  • 无 API 依赖：使用浏览器原生 fetch + DOMParser 解析 RSS XML
  • CORS 代理：默认使用 api.allorgins.win，失败时自动切换其他代理
  • 本地缓存：5 分钟缓存，减少重复请求
  • 懒加载：IntersectionObserver 实现滚动加载
  • 多格式支持：支持 RSS 2.0、RSS 1.0、Atom 格式
